# SE2 - Posicao Diaria

Aplicacao estatica para consulta e acompanhamento de contas a pagar.

## Arquivos principais

- `index.html`
- `app.js`
- `backup.js`
- `datos.js`

## Publicacao no GitHub Pages

1. Crie um repositorio no GitHub e envie os arquivos principais da raiz da pasta.
2. Em `Settings > Pages`, selecione a branch principal e a pasta raiz.
3. Aguarde a publicacao e abra a URL gerada pelo GitHub Pages.

## Observacoes

- A pasta `backups/` e o arquivo `se2_estado.json` sao gerados localmente e nao precisam entrar no repositorio.
- O arquivo `.nojekyll` evita processamento desnecessario do GitHub Pages.
